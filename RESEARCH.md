## 1. デッキレシピ（**サイト間で不一致＝2つの型が存在**）

### 型A: 進化連携型（出典: GameWith [559138](https://gamewith.jp/shadowverse-wb/559138)）
シエテ・サンダルフォン・ルリア・ゴッズレポーターを採用し「進化回数」も勝ち筋にする型。
採用（記事記載, 抽出のため枚数は要現地確認）: 無音の包囲×3 / ルリア×3 / 聴略の諜報兵×3 / 優しき援軍×3 / ゴッズレポーター×2 / 沖田総司×2 / グラン&ジータ×1 / シエテ×3 / ギルダリア×3 / 斬奏の衛生兵×3 / 響爪の分班長×3 / バニー&バロン×3 / サンダルフォン×1 / セザール×3 / マーズ×3 ／ 入替候補: 調伝の連絡兵・不動の将校

### 型B: 連携特化型（出典: Game8 [781940](https://game8.jp/shadowverse-beyond/781940)）
クイックブレイダー・調伝の連絡兵・イドメタ・ナハト&ヴィンセントで連携カウントに寄せた型。
採用: クイックブレイダー×3 / 無音の包囲×3 / 聴略の諜報兵×3 / 優しき援軍×2 / 調伝の連絡兵×3 / ゼタ&ベアトリクス×3 / ギルダリア×3 / 響爪の分班長×3 / 斬奏の衛生兵×3 / イドメタ×2 / セザール×3 / バニー&バロン×3 / ナハト&ヴィンセント×2 / マーズ×3

> **反証メモ**: 両者で1〜7コスト帯の中核（無音の包囲・聴略の諜報兵・優しき援軍・ギルダリア・分班長・衛生兵・セザール・バニー&バロン・マーズ）は**共通**。差分はフィニッシュ補助（型A=シエテ/サンダルフォン、型B=イドメタ/ナハト&ヴィンセント）。
> アプリには「進化連携」の名称に沿って**型A（GameWith）を基準**とし、共通中核カードを軸にマリガン/立ち回りを組む。抽出の枚数は完全一致を保証しないため「参考構築」と明記。

## 3. 戦術・立ち回り（共通見解, 出典: GameWith 559138 / Game8 781940）
- **勝ち筋**: 連携20達成 → ギルダリアのクレスト（自軍フォロワーに突進付与＋進化でスティールナイト2体）＋ バニー&バロン/マーズで打点。型Aは進化6回で**サンダルフォン直接召喚**（毎ターン2×5=10点相当の継続リーサル）。
- **序盤(1-3)**: 低コストを切らさず展開して連携を稼ぎ、**常に盤面有利を維持**（進化ターン前は取り返しが弱いので先手の盤面が命）。
- **中盤(4-6)**: シエテで一斉進化＝テンポ＋進化回数、分班長/衛生兵で横展開＆ドロー補充。
- **終盤(7-)**: セザール(+1/+3守護)で壁＋盤面、マーズ(8)疾走で詰め。**マーズで連携20達成→バニー&バロン/ギルダリア連打で決める**のが基本フィニッシュ。
- **テクニック**: ①ギルダリア打点/処理のため自分のフォロワーをあえてぶつけて破壊数を稼ぐ ②サンダルフォンの5点はランダムなので通す前に相手盤面を除去し顔に寄せる ③シエテは解放奥義で超進化＝進化回数を多く稼ぐ。

## 4. マリガン（共通見解）
- **全対面キープ**: 無音の包囲 / 聴略の諜報兵 / 優しき援軍（序盤の連携＆盤面）。
- **セット条件**: 低コストが確保できれば 斬奏の衛生兵 / 響爪の分班長(5) をキープ可。
- シエテ（型A）: 序盤の動きが揃っていれば1枚キープ。
- 先攻/後攻でのマリガン差は**記事に明記なし**（→アプリでは一般原則で補完: 後攻は+1ドロー/EPを踏まえ受け・重い札を許容、対アグロは軽い盤面札を優先、と推定表記）。

## 5. 対面別（**明確な相性表は未公開のため推定**）
- 環境に多い**ミッド/進化ナイトメア・AFネメシス**に対しては「**序盤の盤面有利を活かす**」のが要（出典要約）。→ 早いテンポ・連携の押し付けで先手を取る。
- ランプドラゴン等の遅い相手には**速度で押し切る**（相手の大型着地前に連携20＆打点）。
- 具体的な有利不利数値は現時点で信頼できる一次データなし → アプリ上は「推定」と明示し、ユーザーが実戦で上書きできる形にする。
